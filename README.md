Подписи полигонов
===================

Данный модуль позволяет создавать подписи для полигонов.
Он работает на базе [JS API Яндекс.Карт](https://tech.yandex.ru/maps/doc/jsapi/2.1/quick-start/tasks/quick-start-docpage/)
